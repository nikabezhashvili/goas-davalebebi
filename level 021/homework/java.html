about Java history

Java შეიმუშავა ჯეიმს გოსლინგმა Sun Microsystems-ში. ის გამოვიდა 1995 წლის მაისში, როგორც Sun-ის Java პლატფორმის ძირითადი კომპონენტი. ორიგინალური და საცნობარო იმპლემენტაციის Java კომპილატორები, ვირტუალური მანქანები და კლასის ბიბლიოთეკები გამოუშვა Sun-მა საკუთრების ლიცენზიებით.

java-ს განვითარება წლების მიხედვით

1991 – დაიწყო პროექტი Oak სახელით (Sun Microsystems, ჯეიმს გოსლინგი). მიზანი იყო სხვადასხვა მოწყობილობაზე გამშვები ენა. 1995 – ენა ოფიციალურად დაერქვა Java და გამოვიდა პირველი ვერსია. მთავარი იდეა: Write Once, Run Anywhere. 1996 – Java 1.0 ვებ-აპლეტები და პლატფორმათაშორისი მუშაობა. 1998 – Java 2 (J2SE 1.2) გამოჩნდა Swing, Collections Framework. 2000 – Java 1.3 გაუმჯობესდა შესრულება და სტაბილურობა. 2002 – Java 1.4 დაემატა assert, რეგულარული გამოსახულებები (regex). 2004 – Java 5 (1.5) დიდი განახლება: Generics, Annotations, Enum, for-each loop. 2006 – Java 6 ვებ-სერვისების და მეხსიერების გაუმჯობესება. 2011 – Java 7 try-with-resources, diamond operator (<>). 2014 – Java 8 ერთ-ერთი ყველაზე მნიშვნელოვანი ვერსია: Lambda expressions, Stream API. 2017 – Java 9 მოდულური სისტემა (Project Jigsaw). 2018 – Java 10 var – ლოკალური ცვლადების ტიპის ავტომატური განსაზღვრა. 2019 – Java 11 (LTS) გრძელვადიანი მხარდაჭერა, უფრო სწრაფი და უსაფრთხო. 2021 – Java 17 (LTS) თანამედროვე ფუნქციები, უკეთესი წარმადობა. 2023–2024 – Java 21 (LTS) Virtual Threads, მაღალი წარმადობა და თანამედროვე სინტაქსი.

Java-ს თავდაპირველი სახელი იყო Oak – მუხის ხის გამო, რომელიც ჯეიმს გოსლინგის ოფისის ფანჯარასთან იდგა. Java-ს ლოგო ყავასთანაა დაკავშირებული – სახელი Java მოდის ინდონეზიური ყავის კუნძულიდან. Java არ იყენებს pointer-ებს პირდაპირ – ეს ზრდის უსაფრთხოებას და ამცირებს შეცდომებს. Minecraft თავდაპირველად Java-ზე დაიწერა – ამიტომაც დღემდე ძალიან პოპულარულია Java Edition. Android-ის პირველი ვერსიები Java-ზე მუშაობდა – აპლიკაციების დიდი ნაწილი ჯერ კიდევ Java/Kotlin-ით იწერება. Java-ს სლოგანია – Write Once, Run Anywhere, რადგან JVM-ზე მუშაობს.
